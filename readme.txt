== Description ==

This plugin makes it simple to gather most of the relevant data you want to keep in your dive log, such as max. depth, bottom time, water temperature, etc.

It has a collection of basic icons to summarise this data and present it in the post. 

You can configure aspects like the units you use (kg./lb., meters/feet), the icons you want to display or not, ect...

== Installation ==

1. Download de Plugin zip file.
1. Unzip and move the folder under your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What if I want to write a post without displaying the dive log? =

You can switch on or off the plugin at post level.
